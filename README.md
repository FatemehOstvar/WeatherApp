A full-stack weather app that lets you search by city and view current conditions. The frontend is built with **Vite**, and the backend serves the production build for deployment.

**Live:** [https://weatherapp-1-zmbe.onrender.com](https://weatherapp-1-zmbe.onrender.com)

---

## ✨ Features

* Displays **weather conditions**
* Metric toggling
* Search For cities
* Sign up + Login + Sign out
* Save city as a logged in user
* Displays the list of cities
* Authentications using JWT tokens
* **Input validation** for cleaner, safer requests
* Responsive UI
  
---

## 🧱 Tech Stack

* **Frontend:** Java script + Html+ Css + Vite (source in `src/`)
* **Backend:** Node.js + Express + PostgreSQL on Neon (source in `server/`)
* **Deployment:** Render
